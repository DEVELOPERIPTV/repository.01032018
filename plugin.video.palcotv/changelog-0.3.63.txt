[B]3.1.15[/B]
Reproducción de películas de la biblioteca de Kodi a través de PalcoTV abriendo un selector con las opciones de un #multi (linkers, conectores y torrents) de una lista M3U local u online.
Nueva sintaxis para poder crear selector de opciones en el skin XML
Recuperado el soporte de listas PLX con paginación, sinopsis e imágenes (logo/background), podcasts y listas de reproducción de podcasts de iVoox (gracias a V1k1ng0 y Aquilesserr), así como nueva sintaxis para búsqueda de torrents (BUM) definiendo qué número mínimo de semillas debe tener cada resultado a mostrar. Se añade además sintaxis (plx:) para abrir listas PLX desde listas M3U.
Corregidos varios bugs en el ScraperX, además de implementar en el menú contextual una opción para agregar películas a la biblioteca de Kodi enlazando con enlaces #multi de una lista local o en línea. Se añade a la configuración varios campos para establecer lista M3U online, APIs de TVDB/TMDB y definir carpeta que servirá de fuente de medios para la biblioteca de Kodi.
Al reproducir un archivo torrent se abre cuadro de opciones con los add-ons de reproducción de torrents instalados. Se añade soporte para Torrentin, Torrenter y YATP.
Actualización de linkers y conectores multimedia gracias a Aquilesserr. No olvides consultar en el foro el estado actual de cada uno de ellos.
Se eliminan varios scripts requeridos para la ejecución de PalcoTV
Recuperada herramienta de fútbolenlatv.com añadiendo soporte de todas las secciones deportivas: Fútbol, Baloncesto, Balonmano, Ciclismo, Golf, Automovilismo, Motociclismo, Fútbol Sala, Fútbol Americano
Pequeños cambios para adaptarse a Kodi 17 (Krypton)
Pequeñas optimizaciones y bugs corregidos
